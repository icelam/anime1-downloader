### 使用教學

#### 安裝 Python
本程式使用 Python 3 進行開發，使用前請先確保 Python 3 已正確安裝。

#### 利用 pipenv 建立 Python 虛擬環境
```bash
# 假設用戶已預先安裝好 pipenv 套件，如未安裝請先運行 pip3 install pipenv
pipenv --python 3.9.9
```

#### 安裝所需套件
```bash
pipenv install --dev
```

#### 在虛擬環境運行主程式
```bash
pipenv run python anime1download
```
