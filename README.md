### 使用教學

#### 利用 Virtualenv 建立 Python 虛擬環境
```bash
# 假設用戶已預先安裝好 virtualenv 套件，如未安裝請先運行 pip3 install virtualenv
virtualenv venv --python=python3
```

#### 啟動虛擬環境及安裝所需套件
```bash
. venv/bin/activate
pip install -r requirements.txt
```

#### 運行主程式
```bash
python src/main.py
```
